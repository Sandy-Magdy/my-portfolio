# Sandy Magdy – Portfolio

Frontend Developer portfolio. Built with React, Vite, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Sections

- **Home** – Hero with name, title, and CTA
- **About** – Summary and location/languages
- **Experience** – DIGILIANS, FMG Agency, Freelance
- **Education** – BIS, Modern Academy
- **Certifications** – DEPI, Route Academy
- **Skills** – Frontend, Backend, Database, Tools
- **Projects** – Daniels, Smart Login Page
- **Contact** – Email, phone, LinkedIn, GitHub
